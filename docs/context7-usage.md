# Context7 MCP usage

Context7 provides **up-to-date Playwright documentation** inside Cursor.

## When to use

- Playwright API questions (`test.step`, locators, fixtures)
- Best practices for POM and assertions
- Debugging strict-mode or timeout issues

## Example prompts

- `use context7: How do I use test.step in Playwright?`
- `use library /microsoft/playwright for locator best practices`

## Setup

See `.cursor/mcp.example.json` — Context7 OAuth URL or API key from [context7.com/dashboard](https://context7.com/dashboard).

Context7 does **not** drive the browser; use **Playwright MCP** for that.
